How to compile hydrogen on linux

These are the instructions for compiling the Hydrogen versions 0.9.6 (based on QT4) on a debian-based distribution. Hydrogen runs on all kinds of linux distribution, but then the package names and installation tools will differ.

Required libraries

  • libqt4-dev
  • libasound-dev
  • libsndfile-dev
  • libjack-dev
  • liblrdf-dev
  • libflac++-dev
  • libtar-dev
  • liblash-dev
  • zlib1g-dev

On a debian based system, you can fetch the mandatory libraries with:

apt-get install git-core cmake libqt4-dev g++ libasound2-dev libjack-dev liblrdf0-dev libflac++-dev libtar-dev libsndfile1-dev

You should use QT 4.6 or higher. In addition, you need a libjack > 1.9.7.

If you want to use lash, install additionally: liblash-dev

If you want to use rubberband, install additionally: librubberband-dev

If you want to use OSS, install additionally: oss4-dev

For portaudio / portmidi: libportaudio-dev libportmidi-dev

1. Get the sourcecode

cd /usr/src git clone git://github.com/hydrogen-music/hydrogen.git

2. Compile and install hydrogen:

cd /usr/src/hydrogen ./build.sh m

3. Build a debian package (optional)

If you're using a debian based system, you can build a debian package. This makes it easier to remove hydrogen from your system.

apt-get install fakeroot debhelper build-essential cd /usr/src/hydrogen/linux fakeroot dpkg-buildpackage

The package appears in /usr/src/hydrogen and can be installed with dpkg (dpkg -i package_name)
